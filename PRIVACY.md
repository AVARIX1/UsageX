\# UsageX – Privacy Policy

© 2024-2026 AVARIX — All Rights Reserved



\## 1. Overview

UsageX is designed to run locally on your device.

It does not collect, store, or transmit personal data to the developer.



\## 2. Data Processed Locally

UsageX may read and display system information such as:



\- CPU usage and frequency

\- Memory usage

\- Disk usage

\- Network statistics

\- System details (hostname, OS version, processes, threads)



This data is processed \*\*locally\*\* and is not sent anywhere unless you explicitly configure an integration.



\## 3. Webhooks \& Integrations

If you configure a Discord webhook or any external integration:



\- UsageX will send selected system information to the endpoint you provide

\- The developer does not receive, store, or access this data

\- You are fully responsible for the security of your webhook URL



\## 4. Local Storage

UsageX may store configuration data such as:



\- Language

\- Theme

\- Webhook URL (optional)

\- Discord User ID (optional)

\- Send interval



All configuration data is stored \*\*locally on your device\*\*.



\## 5. No Telemetry

UsageX does \*\*not\*\* include:



\- analytics

\- tracking

\- telemetry

\- remote logging



\## 6. Third‑Party Policies

If you use Discord or other integrations, your data is subject to their policies.

This Privacy Policy may be updated at any time.

Continued use of the software constitutes acceptance of the updated policy.
